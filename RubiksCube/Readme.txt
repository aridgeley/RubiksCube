The application is written in C# .Net (.Net Framework) and is intended to run within an IDE such as Visual Studio or Rider.
It is intended as a proof of concept. As such, the functionality is limited but it is designed to be readily extendable by another developer, as the 
core design is in place.
The program.cs file to design to be edited so that the number of operations can be varied to give different results (which are printed to a console).
The squares on each face are specified by looking at 'expanded' version of the Rubik cube with all sides laid flat
(as per the example of on https://rubik-cube-solver.com). The initial state of the cube is the same as that on  https://rubik-cube-solver.com.
